# 🧠 CSP Playground

An interactive web application to explore Constraint Satisfaction Problems (CSP) through visual solvers.

## 🚀 Features
- 🔢 Sudoku Solver (Backtracking)
- ♛ N-Queens Visualizer
- 📅 Timetable Generator
- 🌙 Dark/Light Mode
- ✨ Smooth animations & UI

## 🧩 Tech Stack
- HTML, CSS, JavaScript
- Pure frontend (no frameworks)

## 🎯 Concepts Covered
- Variables
- Domains
- Constraints
- Backtracking Algorithm
