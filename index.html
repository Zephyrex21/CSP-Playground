Explore the power of Constraint Satisfaction Problems through interactive solvers for Sudoku, N-Queens, and Timetable Scheduling.

Variables

The unknowns we need to determine. In Sudoku, each empty cell is a variable. In N-Queens, each row's queen position. In scheduling, each subject's time slot.

Domains

The set of possible values for each variable. A Sudoku cell can hold 1-9, a queen can be placed in any column, and a subject can occupy any available slot.

Constraints

Rules that restrict variable assignments. No duplicate numbers in a row, no attacking queens, no teacher conflicts. Constraints define the problem's logic.

Sudoku Solver

Enter a puzzle or load a sample. The backtracking algorithm will find the unique solution respecting all row, column, and block constraints.

How it works

  1. 1Variables: Each of the 81 cells in the 9x9 grid
  2. 2Domains: Digits 1 through 9 for each empty cell
  3. 3Constraints: No duplicate in any row, column, or 3x3 sub-grid
  4. 4Algorithm: Backtracking — try a value, recurse, undo if stuck
Tip: Use only digits 1-9. Leave empty cells blank. Invalid puzzles (duplicate givens) will be detected automatically.
